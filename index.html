NHS trust + ICB data

Public reference dataset for English NHS trusts and integrated care boards.
Last built: 5 June 2026. Refreshed automatically every Saturday.

Four datasets, each available in three formats. XLSX is the most readable in Excel (with sortable columns); CSV is good for quick scripts; JSON is the canonical machine-readable format.

Dataset Entries Canonical Flat Spreadsheet
Trust board-papers URLs 207 JSON (224.2 KB) CSV (159.8 KB) XLSX (58.1 KB)
ICB board-papers URLs 36 JSON (54.0 KB) CSV (37.3 KB) XLSX (18.7 KB)
Trust press & FOI contacts 207 JSON (102.5 KB) CSV (57.8 KB) XLSX (25.3 KB)
ICB press & FOI contacts 36 JSON (17.3 KB) CSV (9.7 KB) XLSX (9.0 KB)
The data is aggregated from publicly-listed information on each organisation's website. Released under CC0 1.0 — use freely. The repository on GitHub has commit history for every change; old URLs pointing at nhs-board-papers-reader (this repo's previous name) still redirect.
